# APASS star catalog (PCC)

This directory holds the bundled APASS DR10 star catalog used by
Polaris's Photometric Color Calibration (PCC) workflow. The
`apass.db` file itself is **not committed to the repo** (the file is
~80 MB and the catalog data has its own attribution requirements);
this README is the placeholder.

## Populating the catalog

Run the download script once on the deployment host:

```bash
python scripts/download-apass.py
```

That fetches the APASS DR10 ASCII slices, filters them to stars with
V magnitude <= 13, and builds a SQLite database with an R*tree
spatial index at `wwwroot/catalogs/apass/apass.db` (~80 MB).

After that, Polaris's `ApassCatalog` service picks the file up
automatically. PCC requests check `IsAvailable` and fail cleanly
with a "run the download script" message if it's missing.

## Alternative catalogs

Polaris's `ApassCatalog` reads the standard schema documented at the
top of `download-apass.py`. If you want to use Gaia DR3 or Tycho-2
instead, build a SQLite file matching that schema and drop it in as
`apass.db` (or change the path in `ApassCatalog.cs`).
